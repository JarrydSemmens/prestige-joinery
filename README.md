# Prestige Joinery

Website for Prestige Joinery — bespoke handcrafted small furniture, made in Carrum Downs, Victoria, Australia.

Plain static HTML/CSS/JS with no build step. See `context/brief.md` for project intent, tech stack and placeholder status.

Local preview: serve the repo root with any static file server, e.g. `python -m http.server`, then open `http://localhost:8000`.
